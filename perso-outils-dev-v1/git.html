GIT

Installation, configuration & mise à jour

Installation

Télécharger et installer Git depuis le site : https://git-scm.com/

Configuration

Ouvrir le terminal Git Bash pour renseigner les paramètres globaux nom d'utilisateur et email :

  • git config --global user.name "nom"
  • git config --global user.email nom@domaine

Vérifier que les paramètres globaux ont bien été pris en compte :

  • git config --global --list

Mise à jour de Git

Vérifier la version de Git dans le terminal Git Bash :

  • git version

Puis télécharger et installer la nouvelle version de Git.

Mise à jour des paramètres globaux

Réitérer les étapes de la configuration de Git.
