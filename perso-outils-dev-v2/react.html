REACT

Installer et démarrer REACT dans un projet

Prérequis

Le gestionnaire npm (Node Package Manager) est nécessaire pour l'installation de REACT dans un dossier projet.
Note : npm fait partie des outils mis en place lors de l'installation de Node.js

Installation

Cibler le répertoire contenant le projet web à l'aide du terminal Git Bash (REACT sera installé dans le dossier - ex: react-install - renseigné dans la ligne de commande) et entrer :

  • npx create-react-app react-install

Démarrer REACT

Cibler Cibler le dossier react-install à l'aide du terminal Git Bash, et entrer :

  • npm start

Mise à jour vers GitHub et clonage

Le dossier node_modules est enregistré dans le fichier .gitignore, il n'est donc pas envoyé vers GitHub lors d'un push du site.

Lorsqu'un projet REACT est cloné dans un nouvel environnement, la commande suivante permet de récupérer le dossier node_module (à l'aide des informations stockées dans le fichier .json) :

  • npm install
